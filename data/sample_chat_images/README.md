# BOND Sample Chat Images

This folder contains sample images for testing the Fish Trap decoy chat system.

## Purpose
- Store sample profile photos for decoy profiles
- Test image loading and display in chat conversations
- Provide visual assets for automated testing

## File Naming Convention
- Use descriptive names: `decoy_name_action.jpg`
- Examples:
  - `priya_sharma_smiling.jpg`
  - `arjun_patel_hiking.jpg`
  - `sneha_gupta_coffee.jpg`

## Image Requirements
- Format: JPG, PNG, or WebP
- Size: 500x500px minimum (square aspect ratio preferred)
- Quality: High resolution for good display
- Content: Appropriate for dating app (no explicit content)

## Usage
Images in this folder will be referenced by the decoy profile seed data and used in automated testing of the chat system.
